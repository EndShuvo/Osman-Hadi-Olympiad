SSLCommerz (প্রাথমিক)

bKash API (বিকল্প)
Merchant Checkout API। ইউজার bKash অ্যাপ দিয়ে সরাসরি পেমেন্ট করতে পারবে।
Nagad API (বিকল্প)
নগদ মার্চেন্ট API। ইউজার নগদ অ্যাপ/ USSD দিয়ে পেমেন্ট।

অ্যাডমিন ক্যাশ/ব্যাংক ডিপোজিট হাতে নিয়ে ম্যানুয়ালি সিস্টেমে এন্ট্রি করতে পারবে (ব্যাকঅফিসের জন্য)।

⚠️ পেমেন্ট ফ্লো — কীভাবে কাজ করবে?

  1. ইউজার রেজিস্ট্রেশন ফর্ম পূরণ করে → পেমেন্ট পেজে যায়
  2. ফি অটো ক্যালকুলেট হয় (বয়স ≤১২ = ১৮৳, >১২ = ৩৬৳)
  3. SSLCommerz Initiate API Call → পেমেন্ট URL তৈরি হয়
  4. ইউজার bKash/Nagad/Card দিয়ে পেমেন্ট করে
  5. Success URL এ ফিরে আসে → Backend IPN verify করে
  6. পেমেন্ট ভ্যালিডেট হলে স্ট্যাটাস "Paid" + অটো এসএমএস যায়
  7. ফেইল্ড হলে "Retry Payment" বাটন দেখায়
